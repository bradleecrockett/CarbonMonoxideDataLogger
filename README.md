# Carbon Monoxide Data Logger

Create an arduino device to monitor Carbon Monoxide levels in a mechanic garage.

* Use an MQ-7 gas sensor.
  * Note, this requires 5volts to operate.
* Should be as small and portable as possible.
  * If possible, use a battery so the device coud be worn or carried by a mechanic.
  * Arduino 101, Arduino Pro mini, Microbit (3v).
* Must take readings at least one every 15 minutes.
* Readings should track the ppm of CO.
* Should also measure and log the temperature, humidity and time of day with each measurement.
